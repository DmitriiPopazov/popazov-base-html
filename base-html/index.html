HTML - документ - это обычный текстовый доумент, может быть создан как в обычном текстовом редакторе (блокнот), так и в специализированном, с подсветкой кода (Notepad++, Visual Studio Code и т.п.). HTML - документ имеет расширение .html.

HTML - документ состоит из дерева HTML - элементов и текста. Каждый элемент обозначает в исходном документе начальным (открывающим) и конечным (закрывающим) тегом (за редким исключением).

Начальный тег показывает, где начинается элемент,конечный - где заканчивается. закрывающий тег образуется путём добавление слэша / перед именем тега: ‹имя тега› ... ‹/имя тега›. закрывающим тегами находится содержимое элемента - контент.

Например:

  1. ‹p› Контент ‹/p› - создаёт параграф
  2. ‹b› Контент ‹/b› - Делает полужироное начертание
  3. ‹i› Контент ‹/i› - Курсивное начертание
